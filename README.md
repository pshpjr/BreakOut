# BreakOut

BreakOut은 테트리스 99를 보고 시작하게 된 프로젝트 입니다. 99명의 플레이어가 동시에 벽돌깨기를 진행하며 목숨 3개를 모두 잃을 경우 탈락하게 됩니다.
최후의 1인은 승자가 됩니다. 

## 목적
게임 서버 프로그래밍 공부를 위한 것으로 멀티스레딩, IOCP를 중점적으로 공부합니다.

## 목표 기능
